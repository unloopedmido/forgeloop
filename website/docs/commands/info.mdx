## Usage

```bash
forgeloop info [--dir ./project]
```

Prints a banner with the resolved directory, then:

- **Project** — `manifest.projectName`
- **Language** — `ts` | `js`
- **Preset** — `basic` | `modular` | `advanced`
- **Framework** — `discord.js` (as stored in manifest)
- **Package manager**
- **Docker** / **CI** — enabled or disabled
- **Database** — `none`, or `sqlite via prisma` / `postgresql via prisma`, etc.
