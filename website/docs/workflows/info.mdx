import { CliExample } from '@site/src/components/docs/CliExample';

`forgeloop info` loads `forgeloop.config.mjs` and prints a short summary: project name, language, preset, package manager, Docker/CI flags, and database (provider + ORM or none).

<CliExample
  title="Example (illustrative)"
  command="npx forgeloop info"
  output={`┌──────────────────────────────────────────────┐
│ ForgeLoop info                               │
└──────────────────────────────────────────────┘
  /path/to/my-bot

  • Project          my-bot
  • Language         ts
  • Preset           modular
  • Package manager  npm
  • Docker           disabled
  • CI               disabled
  • Database         none`}
/>

Use **`--dir`** to point at another project root.

Full reference: [`forgeloop info`](../commands/info.mdx).
